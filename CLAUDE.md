# 絶対に守るべきルール

以下のルールを **絶対に** 守ること。

## 1. Plan Mode の使用

コードを書く前に、 **必ず Plan Mode で起動** し、ユーザーとプランを練ること。

実装の方向性をユーザーに確認し、合意を得てから作業を開始すること。

## 2. 音声通知の実行

以下のようなユーザーの操作を待つタイミングで **必ず音声を鳴らす** こと。

- コマンドの実行許可を得るとき
- プランニングが完了したとき
- コーディングが完了したとき

音声を鳴らすには以下の MCP サーバーを使用すること。

| MCP サーバー |   ツール   |
| :----------: | :--------: |
|  play-sound  | play_glass |

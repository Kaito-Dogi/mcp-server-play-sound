# 絶対に守るべきルール

以下のルールを **絶対に** 守ること。
ルールの **優先順位は平等** であり、全てを厳密に守ること。

1. Plan Mode の使用
2. 音声通知の実行

## 1. Plan Mode の使用

コードを書く前に、 **必ず Plan Mode で起動** し、ユーザーとプランを練ること。

実装の方向性をユーザーに確認し、承認を得てから作業を開始すること。

## 2. 音声通知の実行

以下のような **ユーザーの操作を待つタイミング** で **必ず音声を鳴らす** こと。

- AskUserQuestion ツールを使用してユーザーに質問するとき（質問の直前に音声を鳴らす）
- ExitPlanMode ツールを使用してプランを提示するとき（プラン提示の直前に音声を鳴らす）
- Bash などのコマンドを実行するための承認を得るとき
- コーディングが完了したとき

音声を鳴らすには以下の MCP サーバーを使用すること。

| MCP サーバー |   ツール   |
| :----------: | :--------: |
|  play-sound  | play_glass |
